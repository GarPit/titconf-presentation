# Установка

### Standalone Browser Version

```bash
$ git clone git://github.com/Shopify/batman.git
```

### Node.js

```bash
$ npm install -g batman
```

```bash
$ cd ~/projects
$ batman new my_app
```

```bash
$ cd my_app
$ batman server # http://localhost:1047
```
